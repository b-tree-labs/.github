B-Tree Labs builds data infrastructure, machine learning for research,
and the substrate composable agentic systems run on — memory, federation,
classification, governance — and publishes the research that motivates
our designs.

## Active products

| Project | What it does | Status |
|---|---|---|

| **Axiom** | Composable, extensible, LLM-agnostic agentic operating system — unified personal and organization memory, federation, and publishing | In development |
